# Demand Prediction Model
Demand Prediction ML Model that uses CatBoost to predict the product demand using customer and product features. Utilizes Streamlit to present a web-based UI where user can test out the model as well. 

### Model Evaluation
**Model Used:** CatBoost
Achieved MAE of 12.84, which is a 59% improvement over the naive baseline.
Along with target's mean as ~104, it makes model's prediction off by ~12% on average.

### Dataset Credit
**Dataset:** [Retail Store Inventory and Demand Forecasting](https://www.kaggle.com/datasets/atomicd/retail-store-inventory-and-demand-forecasting)")
**Author:** [Wavelet](https://www.kaggle.com/atomicd)")
